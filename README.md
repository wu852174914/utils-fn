### 1. 简介

本项目是js常用工具函数 集合，包含了常用的工具函数，如：

- 类型判断函数
- 数值格式化取整函数
- 防抖函数
- 节流函数
- 文件下载函数
- 数组去空函数
- 数组随机排序函数
- 数组随机取值
