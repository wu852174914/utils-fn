### 1. 简介

本项目是js常用工具函数 集合，包含了常用的工具函数，如：

- 类型判断函数
- 数值格式化取整函数
- 防抖函数
- 节流函数
- 文件下载函数
- 数组转为树状数据函数
- 查找树状数据中的节点函数
- 树状结构扁平化函数
- 对象合并函数
- 深拷贝函数
- url中获取参数函数
- 正则验证函数
  - 手机号码验证
  - 邮箱验证
  - 身份证验证
